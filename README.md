# Calculadora IMC

Este projeto é uma calculadora simples de IMC (Índice de Massa Corporal) feita em JavaScript.

## Como funciona

O IMC é calculado pela fórmula:

```
IMC = peso / (altura * altura)
```

- **peso**: em quilogramas (kg)
- **altura**: em metros (m)

## Como executar

1. Instale o [Node.js](https://nodejs.org/).
2. No terminal, navegue até a pasta do projeto.
3. Execute:
   ```
   node calculadora-imc.js
   ```

## Exemplo de uso

O script pode ser adaptado para receber dados do usuário via terminal ou navegador.
